## 👩‍💻 Features

- [x] Using Typescript in src / testing
- [x] Developer friendly dev-environment with linting & hot-reload
- [x] Testing using Jest/Supertest
- [x] Advanced logging using Winston/Morgan

## 🙏 Installation

```bash
npm install
```

## 🏃 Running the app

```bash
# development
npm run start

# watch mode
npm run start:dev

# production mode
npm run start:prod
```

## ✔️ Test

```bash
# unit tests
npm run test

# e2e tests
npm run test:e2e

# test coverage
npm run test:cov

# unit tests with watcher
npm run test:watch
```
